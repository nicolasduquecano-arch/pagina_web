1. Operadores Aritméticos
Se utilizan para realizar operaciones matemáticas entre valores o variables.

 Suma
   +
 Resta
   -
Multiplicación
   *
/  División
% Módulo (residuo de una división)
**  Potencia
++  Incremento (aumenta en 1)
--  Decremento (disminuye en 1)

Ejemplo:
let resultado = 5 + 3;

2. Operadores de Asignación
Se usan para asignar valores a las variables.

=  Asignación simple
+=  Suma y asigna
-=  Resta y asigna
*=  Multiplica y asigna
/=  Divide y asigna
%=  Módulo y asigna
**=  Potencia y asigna

Ejemplo:
let a = 10;
x += 5;

3. Operadores de Comparación
Permiten comparar valores y devuelven un resultado booleano (true o false).

== → Igual (solo valor)
=== → Igual estricto (valor y tipo)
!= → Diferente
!== → Diferente estricto

> Mayor que
< = Menor que
= > Mayor o igual
<=  Menor o igual

4. Operadores Lógicos
Se utilizan para combinar condiciones o expresiones booleanas.

&& → Y (y lógico)
|| → O (o lógico)
! → NO (negación)

Ejemplo:
let edad = 20;
edad > 18 && edad < 30;
